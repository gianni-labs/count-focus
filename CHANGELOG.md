# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog 1.1.0](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Changed

- Clarified the README value proposition and installation path in English and Spanish.

## [0.3.0] - 2026-07-10

### Added

- Full four-round Pomodoro cycle with short and long breaks.
- Count-up mode with an optional goal and completion alert.
- `--exec` hook to run a shell command when a timer or goal completes.
- Spanish README.

## [0.2.0] - 2026-07-02

### Added

- Named presets and user-configurable preset durations.
- `--title` and `--until` timer options.
- Pause and resume controls, a progress bar, terminal bell, and `--version`.
- CI checks for formatting, vetting, tests, and builds.

## [0.1.1] - 2026-07-02

### Added

- MIT license and terminal demo GIF.

### Changed

- Homebrew installation now builds from versioned release tarballs.

## [0.1.0] - 2026-07-01

### Added

- Initial release of Count Focus, a responsive terminal focus timer.
